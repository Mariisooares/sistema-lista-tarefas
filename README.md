## Descrição do projeto

Esse projeto tem como objetivo desenvolver um Sistema de Lista de Tarefas (ToDo List), aplicando na prática os conceitos aprendidos na disciplina de Fundamentos de Programação Web.
O sistema foi desenvolvido utilizando HTML5, CSS3 e JavaScript puro, seguindo os requisitos propostos na atividade acadêmica, que incluem cadastro, visualização, edição, exclusão e armazenamento local das tarefas.
O projeto foi estruturado com foco em uma interface intuitiva, moderna e responsiva, utilizando manipulação do DOM e persistência dos dados por meio do LocalStorage do navegador. 

## 📌 Funcionalidades implementadas
- Cadastro de tarefas
- Busca por título
- Filtro por prioridade
- Filtro por status
- Alteração de status
- Edição
- Exclusão
- Persistência com LocalStorage

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript
- GitHub
- GitHub Pages
- Vs Code

Abra o arquivo `index.html` no navegador.
